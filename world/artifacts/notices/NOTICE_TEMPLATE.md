# Official Public Notices — Standard Notice Form

> In-universe document. Ref: `OPN-TEMPLATE-0001`. Every Public Notice Daily Item should be writable as an instance of this form.

```
OFFICIAL PUBLIC NOTICE
Reference: OPN-[####]
Issued: [date]
Distribution: [see STYLE_GUIDE.md — usually "Anyone Who Happens To Be Looking"]

Residents are reminded that [matter of no urgency].

[Optional secondary clause, delivered with equal gravity regardless
of the first clause's actual importance.]

— Issued Under Authority Unconfirmed
```

## Standing Rules

- The word "residents" is never qualified ("some residents," "affected residents"). All notices address all residents equally, regardless of relevance.
- Notices never apologize for the inconvenience, because no inconvenience is ever acknowledged as such.
- A notice may reference a prior notice by reference number but must never explain why the prior notice is relevant.
